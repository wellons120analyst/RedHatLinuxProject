
# RED HAT LINUX PROJECT

## Objective

Configure and secure Linux environment by leveraging command-line tools to create and manage users and groups, enforce password and access policies, assign administrative privileges, and control file ownership to support strong IAM practices.

### Skills Learned

- User and group management
- Role-based access control
- Password policies & account aging
- Least privilege enforcement
- Privilege escalation control
- Managing service accounts


### Tools Used

Red Hat Enterprise Linux
